What this is

An independent end-of-pipeline check that the Quanta multimodal brain-age stack generalises to held-out subjects from the same cohort. The 86-subject test set is fully disjoint from the 341 training subjects, and auxiliary out-of-fold scalar predictors (DTI ROI / DTI voxel ridge / rs-fMRI FC) were regenerated under the frozen split so no test labels reach the held-out rows.

Cohort & model

  • Cohort. 427 healthy adults, age 19–80, TCNL Quanta multimodal ageing study.
  • Modalities. behaviour · EEG · T1-w MRI morphometry · DTI ROI + voxel · compositional features · rs-fMRI Schaefer-400 FC · deep features (SFCN, Pyment).
  • Stack. per-modality Ridge / KRR-RBF / LGBM-L2 / SVR-RBF / LGBM-quantile base learners feeding a RidgeCV meta-learner with a residual Ridge correction.
  • Split. 341 train / 86 held-out, age × sex stratified, frozen before the final run.
